# Clean-Anime-Image

> A Simple and Efficient Anime Image Restoration Tool

## Installation

```shell
git clone https://github.com/AsherJingkongChen/clean-anime-image.git && \
cd clean-anime-image && \
./install.sh
```

## Usage

```shell
clean_anime_image -h
```

```shell
clean_anime_image -i inputs/ -o outputs/ -s 2.0
```
